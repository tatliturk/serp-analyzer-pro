# SERP Analyzer Pro

A Chrome extension that adds instant SEO insights directly on Google search results.

## Features

- **Position badge** — See the exact rank of every organic result
- **Title length** — Color-coded check against the optimal 50–60 character range
- **Meta description** — Instant check against the 120–160 character sweet spot
- **Domain info** — See the domain name for every result at a glance

## Color Guide

- 🟢 Green — Optimal range, well optimized
- 🟡 Yellow — Too short, needs improvement
- 🔴 Red — Too long, over the limit
- 🔵 Blue — Informational

## Installation

Install from the [Chrome Web Store](https://chrome.google.com/webstore)
